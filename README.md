# SydneyViya-SongList

這是提供演唱者以EXCEL整理自己歌單，透過網頁給大家更方便的瀏覽尋找想要的歌

搜尋模式可透過任何關鍵字、標籤、歌手快速找到想聽的歌。

此做模板歌單是Vtuber[希妮薇婭SydneyViya Ch.](https://www.youtube.com/@Sydney_Viya_0525/featured)
